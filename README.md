# assets_management

Test code for using the NeMO assets SQL tables.

## Purpose

This repository is a starter workspace for experimenting with SQL queries, documenting findings, and adding lightweight helper scripts related to NeMO asset data.

## Initial Layout

- `src/sql/` - SQL scripts and query drafts
- `scripts/` - Utility scripts for local workflow
- `docs/` - Notes, assumptions, and findings

## Quick Start

1. Add SQL scripts in `src/sql/`.
2. Keep experiment notes in `docs/`.
3. Add helper automation in `scripts/` when needed.
